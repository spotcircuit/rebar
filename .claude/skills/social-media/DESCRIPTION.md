# Social-media skills

Distribution, generative-engine optimization, and launch-cadence playbooks for short-form posts, threads, and multi-channel campaigns. Load when output target is a social platform or AI-search citation surface.

## Skills here

- `ai-seo/` — generative engine optimization for ChatGPT, Perplexity, and Claude citations. Load before publishing any post intended to be cited by an AI assistant.
- `launch-strategy/` — phased GTM (pre-launch → launch-week → sustaining). Load when planning a product/feature launch or coordinating a multi-day campaign.

## Related (lives elsewhere)

- `content/copywriting/` — hooks, headlines, CTAs. Use it for tweet hooks, LinkedIn opens, post titles.
- `apps/cross-post/` — the rebar-specific pipeline that publishes to Bluesky, LinkedIn, Mastodon, X, etc.
- `apps/social-scout/` — Scout's reply-generation flow.

## When to load this category

- Drafting a post or thread for X/LinkedIn/Bluesky/Mastodon
- Planning a launch sequence (pre-launch teasers, launch-week cadence, sustaining)
- Optimizing a page or post to be cited by ChatGPT/Perplexity/Claude
